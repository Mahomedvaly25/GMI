# MGI Lda. - Mozambique Group Investments

Corporate Landing Page developed for Mozambique Group Investments, Lda. (MGI). High-performance, single-page application built with React, Next.js, Tailwind CSS, and Framer Motion. Engineered for elite logistics, supply chain representation, and heavy machinery operations across the SADC region. Optimized for responsive layouts, strict aduanas compliance, and zero-error TypeScript compilation.

## Technical Stack

- **Core**: React 19 / TypeScript
- **Styling**: Tailwind CSS
- **Bundler & Dev Server**: Vite
- **Animations**: Framer Motion (`motion/react`)

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```
   The application will be served at `http://localhost:3000`.

### Scripts

- `npm run dev`: Starts the Vite development server.
- `npm run build`: Compiles and packages the application for production inside the `dist/` directory.
- `npm run lint`: Runs the TypeScript compiler to ensure strict type-safety compilation.
